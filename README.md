# Students Social Media Addiction - Exploratory Data Analysis

## Project Overview
Performed exploratory data analysis (EDA) on a student social media addiction dataset using Python to identify behavioral patterns, screen-time trends, and factors influencing social media dependency.

## Objectives
- Analyze student social media usage patterns
- Identify addiction-related behavioral trends
- Study relationships between screen time, sleep, and academic performance
- Detect correlations and outliers
- Generate meaningful insights through visualizations

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset Features
- Age
- Gender
- Daily screen time
- Most used platform
- Sleep hours
- Academic performance
- Addiction level

## Exploratory Data Analysis Performed
- Missing value analysis
- Correlation analysis
- Distribution analysis
- Outlier detection
- Platform usage comparison
- Screen time analysis
- Visualization of behavioral trends

## Visualizations
- Heatmaps
- Histograms
- Box plots
- Count plots
- Scatter plots
- Bar charts

## Key Insights
- Identified relationships between screen time and addiction levels
- Observed trends in social media usage among students
- Analyzed the impact of social media habits on sleep and academics
- Found highly correlated behavioral features

## How to Run

```bash
pip install -r requirements.txt
jupyter notebook
```
